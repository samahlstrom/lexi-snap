                        Dict-to-Anki
          Create Anki flashcards from anywhere with a hotkey!
================================================================================

WHAT IS THIS?
-------------
Dict-to-Anki lets you highlight any word on your computer (in browsers, PDFs,
documents, anywhere!) and instantly create an Anki flashcard with definitions.

Just press Ctrl+Alt+D on any selected text!


QUICK START
-----------
1. Install Anki from: https://apps.ankiweb.net/

2. Install AnkiConnect add-on:
   - Open Anki
   - Tools â†’ Add-ons â†’ Get Add-ons
   - Enter code: 2055492159
   - Restart Anki

3. Start Dict-to-Anki:
   - Double-click DictToAnki-Service.exe
   - You'll see: "Dict-to-Anki Hotkey Service started"

4. Use it:
   - Open Anki (keep it running)
   - Highlight any word
   - Press Ctrl+Alt+D
   - Select a deck
   - Done!


AUTO-START (OPTIONAL)
---------------------
To make Dict-to-Anki start automatically when Windows starts:

1. Press Windows key + R
2. Type: shell:startup
3. Press Enter
4. Copy DictToAnki-Service.exe into that folder
5. Restart Windows
6. Service now starts automatically!


USAGE EXAMPLE
-------------
You're reading: "The ubiquitous smartphone has transformed communication."

1. Highlight "ubiquitous"
2. Press Ctrl+Alt+D
3. Select "English Vocabulary" deck
4. Card created with definitions!
